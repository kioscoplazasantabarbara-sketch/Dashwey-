DASHWEY v8.6.4
==============
Fecha: 2026-03-19
Desde: v8.4 instalada en dispositivo

IMPORTANTE — FORZAR ACTUALIZACIÓN:
Esta versión es mayor que v8.5.9 (la que mostraba el checker).
Al subir a GitHub, el sistema detectará automáticamente la nueva versión.

FIXES INCLUIDOS:

1. AUTH / LOGIN
   - file://: salta auth directamente (modo local sin bloqueo)
   - GitHub Pages: login solo si usuario es anónimo (no registrado)
   - Timeout 5s: si Firebase no responde → modo local automático
   - authLogin/Register: dispara sync inmediato tras login exitoso
   - Botón "Continuar sin cuenta" más visible

2. BUSCADOR — Sugerencias
   - Los resultados ya no aparecen desde la navbar inferior
   - Aparecen en cascada debajo del cajón de búsqueda
   - Posicionadas con CSS (absolute dentro de wrapper relativo)
   - Sin posicionamiento JS frágil

3. BOTÓN PRINCIPAL — Estados claros
   Estado 0: "Ver catálogo" → abre el catálogo
   Estado 1 (sin items): "Cerrar catálogo"
   Estado 1 (con items): "Aceptar (N art.)" → confirma + popup análisis
   Estado 2: "Añadir más" (azul)

4. POPUP AL ACEPTAR
   - Score del pedido, KPIs, alertas, recomendaciones IA
   - Botón "Ver resumen" descarga .txt con análisis completo

5. AUDITORÍA v8.6.0
   - Eliminado código legacy (ns-overlay, clearSearch, onSearch)
   - Stubs limpios para funciones obsoletas
   - 0 console.log en producción

PARA FORZAR ACTUALIZACIÓN EN EL DISPOSITIVO:
1. Sube los 3 archivos a GitHub
2. Abre la app → Settings → Actualizaciones → Comprobar
3. Pulsa "Actualizar ahora"
4. O: borrar caché del sitio en Chrome → volver a abrir
