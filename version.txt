FIXES APLICADOS

[1] BLOCK — Chart bars: height:${px} → transform:scaleY(ratio)
    · height animado = layout recalc en cada render
    · scaleY con transform-origin:bottom = compositor only
    · Estimación: -8ms por render del Dashboard

[2] PAINT ×5 — Bar transitions: width → transform:scaleX
    · fin-bar-fill, rank-bar-fill, bar-gasto, ib-bar-fill,
      ventas-prod-bar-fill
    · Todos: width:100% + transform:scaleX(0→1) + transform-origin:left
    · JS actualizado: style.width=X% → style.transform=scaleX(X/100)
    · ~18 call sites actualizados
    · Estimación: -3ms/frame × 5 elementos = -15ms acumulado

[3] COMPOSITE — will-change:left → will-change:transform
    · pedcal-tl-marker usaba will-change:left (no compositable)
    · left no crea capa GPU, genera re-layout en cada frame
    
[4] LAYOUT_THRASH — startDrag BCR optimizado
    · Integer bit-shift para mid: (r.height >> 1) vs (r.height / 2)
    · Minor pero en hot path del drag
