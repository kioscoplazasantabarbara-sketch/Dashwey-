CHANGELOG v1.2.29-dev — Sistema Global de Manejo de Errores
=============================================================

[FEAT-1] window.onerror implementado (reemplaza listener básico)
  - Captura: mensaje, fuente, línea, columna, objeto Error
  - Filtra errores de recursos externos (img, script) — no spam al usuario
  - Llama a _tryRecoverUI() automáticamente antes de notificar
  - Solo muestra toast si el error es de código propio de la app

[FEAT-2] Try/catch en funciones críticas de navegación y render
  - goTab()          — navegación entre tabs nunca puede congelar la app
  - openFs()         — apertura de full-screen sheet protegida
  - closeFs()        — cierre de sheet protegido (overflow body siempre restaurado)
  - openModal()      — apertura de modal protegida
  - closeModal()     — cierre de modal protegido
  - openSettings()   — drawer de ajustes protegido
  - closeSettings()  — cierre de drawer protegido
  - renderOrden()    — render ALM protegido (ya existía en dash/tpv)

[FEAT-3] Fallback UI — recuperación automática (_tryRecoverUI)
  - Restaura document.body.overflow si quedó bloqueado por sheet a medias
  - Cierra overlays huérfanos (fs-overlay, modal-overlay sin contenido)
  - Garantiza visibilidad de bottom-tabs
  - Se invoca desde onerror Y desde unhandledrejection

[FEAT-4] Logger centralizado window._DashweyLog
  - _DashweyLog.error(tag, msg, extra) — errores
  - _DashweyLog.warn(tag, msg, extra)  — advertencias
  - _DashweyLog.info(tag, msg, extra)  — info
  - _DashweyLog.dump()                 — devuelve historial de últimos 50 errores
  - Accesible desde consola Android para depuración: window._DashweyLog.dump()

[FEAT-5] Toast throttle — máx. 1 aviso de error cada 4 segundos
  - Evita spam de toasts si hay un error recurrente
  - Solo errores de código propio generan toast (no recursos)

[FIX] unhandledrejection mejorado
  - Extrae mensaje del Error object o string correctamente
  - Incluye stack trace en el log cuando está disponible
  - Llama a _tryRecoverUI() en rejections también
